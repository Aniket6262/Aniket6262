Finishing my MS in Software Engineering at Arizona State University (May 2026). My work spans backend infrastructure, LLM fine-tuning, and the evaluation + security testing that decides whether a model is actually ready to ship. That combination of ML depth and software-engineering discipline is where I do my best work.

Python PyTorch DistilBERT
